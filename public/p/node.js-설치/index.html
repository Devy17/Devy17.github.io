# Chocolatey 설치
powershell -c "irm https://community.chocolatey.org/install.ps1|iex"

# Node.js 설치
choco install nodejs --version="22.18.0"

# Node.js 버전 확인
node -v # Should print "v22.18.0".

# npm 버전 확인
npm -v # Should print "10.9.3".

# nvm 설치
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash

# shell 재시작
\. "$HOME/.nvm/nvm.sh"

# Node.js 설치
nvm install 22

# Node.js 버전 확인
node -v # "v22.18.0".
nvm current # "v22.18.0".

# npm 버전 확인
npm -v # "10.9.3".

npm, yarn

npm은 Node.js의 기본 패키지 관리자로, 자바스크립트 패키지의 설치, 업데이트, 삭제 등을 관리합니다.
yarn은 npm의 대안으로 나온 패키지 관리자로, 빠른 설치 속도와 안정적인 버전 관리가 특징입니다.
